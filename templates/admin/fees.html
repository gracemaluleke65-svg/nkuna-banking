About Fee Configuration

Configure how fees are calculated for different transaction types in the system.

How Fees Work
  • Fees are calculated as a percentage of the transaction amount
  • The minimum fee ensures a base charge for small transactions
  • The maximum fee caps charges for large transactions
  • Fees can be enabled/disabled without deleting configuration
Important Notes
  • Changes apply immediately to new transactions
  • Existing transactions are not affected by fee changes
  • Always test fee changes with sample amounts
  • Consider user impact when changing fees
